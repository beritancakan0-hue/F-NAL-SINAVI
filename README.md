# FİNAL-SINAVI


Bu proje, **Bootstrap 5** kullanılarak geliştirilmiş modern ve responsive bir **spor salonu (Sports Club) tanıtım web sitesidir**.  
CSS dosyası kullanılmadan, yalnızca Bootstrap sınıfları ile tasarlanmıştır.
Tam responsive (mobil, tablet, masaüstü uyumlu)
 Bootstrap 5 ile modern tasarım
 Navbar ile tek sayfa (one-page) yapı
 Hero alanında arka plan görseli ve karartma efekti
 Hizmetler bölümü (Fitness, Kick Boks, Yüzme)
 Eğitmenler tanıtım alanı
 İletişim bilgileri bölümü
 Koyu ve açık alanların dengeli kullanımı
KULLANILAN TEKNOLOJİLER : *HTML5**
- **Bootstrap 5.3.2**
**SİTE LİNKİ** :https://beritancakan0-hue.github.io/F-NAL-SINAVI/
